# Happiness & Economy Analysis (2020–2024)

This project explores the relationship between global happiness and economic indicators using data from the World Happiness Report (2020–2024) and World Bank GDP per capita (2024).
The tutorial demonstrates how to build a complete, reproducible data science workflow from data collection and cleaning to exploratory analysis, hypothesis testing, and machine learning modeling.


📊 What the Project Does
- Merges happiness data (2020–2024) with GDP per capita
- Cleans and preprocesses the dataset
- Performs exploratory data analysis with clear visualizations
- Runs hypothesis tests (Pearson correlation, ANOVA)
- Builds Linear Regression and Random Forest models
- Creates an interactive Folium world map of happiness scores


🔎 Key Findings
- GDP per capita shows the strongest positive correlation with happiness
- Social support and healthy life expectancy are major contributors
- Generosity shows no significant relationship
- Random Forest identifies GDP as the most important predictor


🛠 Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Folium for interactive mapping
- Jupyter Notebook


📚 Data Sources
- World Happiness Report (2020–2024)
- World Bank Open Data — GDP per capita
